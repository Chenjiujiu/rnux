## Loading 加载组件

### 继承自 Spinner, 支持Spinner组件所有属性

| 属性           | 描述                  | 值            | 类型      | 默认值   |
|--------------|---------------------|--------------|---------|-------|
| cover        | 撑满父容器               | true / false | boolean | false |
| fullscreen   | 全屏                  | true / false | boolean | false |
| modalVisible | 模态框可见<br/>兼容模态框关闭动画 | true / false | boolean | true  |
