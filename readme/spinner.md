## Spinner 旋转加载组件

| 属性       | 描述                                        | 类型                                 | 默认值      |
|----------|-------------------------------------------|------------------------------------|----------|
| size     | 大小 `small`/`medium`/`large`               | string                             | `medium` |
| color    | 圈圈颜色 `black`/`primaryLight`/`#667788`/... | [colors key](./theme.md)  / string | `black`  |
| duration | 一圈时间 ms                                   | number                             | 800      |

> theme.spinner 中可自定义 size 与 borderWidth
