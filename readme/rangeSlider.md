## RangeSlider 滑动选择器

| 属性           | 描述                     | 类型                            | 默认值   |
|--------------|------------------------|-------------------------------|-------|
| min          | 最小值                    | number                        | 0     |
| max          | 最大值                    | number                        | 100   |
| step         | 步长                     | number                        | 1     |
| defaultValue | 默认值(非受控)               | [number, number]              | [0,0] |
| onChange     | 同 RN Pressable onPress | (val: [number, number]): void | -     |
